# [Regressão Logística do dataset LIAR](./main.py)

https://paperswithcode.com/dataset/liar

## Instalação

Requer-se Python 3.9 ou superior

Utilizando pip:
```
pip install -r requirements.txt
```

Utilizanto Poetry:
```
poetry env use <executável do python>
```

## Uso

```
python -m liar_logreg
```
